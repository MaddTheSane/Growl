GrowlAppBridgeDefinesCarbon.h


Abstract

Defines all the notification keys

Discussion

Defines all the keys used for registration and growl notifications.

#defines

GROWL_APP_NAME

#define GROWL_APP_NAME CFSTR(
    "ApplicationName")
Discussion

The name of your application

GROWL_APP_ICON

#define GROWL_APP_ICON CFSTR(
    "ApplicationIcon")
Discussion

The TIFF data for the default icon for notifications (Optional)

GROWL_NOTIFICATIONS_DEFAULT

#define GROWL_NOTIFICATIONS_DEFAULT CFSTR(
    "DefaultNotifications")
Discussion

The array of notifications to turn on by default

GROWL_NOTIFICATIONS_ALL

#define GROWL_NOTIFICATIONS_ALL CFSTR(
    "AllNotifications")
Discussion

The array of all notifications your application can send

GROWL_NOTIFICATIONS_USER_SET

#define GROWL_NOTIFICATIONS_USER_SET CFSTR(
    "AllowedUserNotifications")
Discussion

The array of notifications the user has turned on

GROWL_NOTIFICATION_NAME

#define GROWL_NOTIFICATION_NAME CFSTR(
    "NotificationName")
Discussion

The name of the notification. This should be human-readable as it's shown in the prefpane

GROWL_NOTIFICATION_TITLE

#define GROWL_NOTIFICATION_TITLE CFSTR(
    "NotificationTitle")
Discussion

The title to display in the notification

GROWL_NOTIFICATION_DESCRIPTION

#define GROWL_NOTIFICATION_DESCRIPTION CFSTR(
    "NotificationDescription")
Discussion

The contents of the notification

GROWL_NOTIFICATION_ICON

#define GROWL_NOTIFICATION_ICON CFSTR(
    "NotificationIcon")
Discussion

The TIFF data for the notification icon (Optional)

GROWL_NOTIFICATION_APP_ICON

#define GROWL_NOTIFICATION_APP_ICON CFSTR(
    "NotificationAppIcon")
Discussion

The TIFF data for the application icon (Optional)

GROWL_NOTIFICATION_PRIORITY

#define GROWL_NOTIFICATION_PRIORITY CFSTR(
    "NotificationPriority")
Discussion

The priority of the notification from the preference pane

GROWL_NOTIFICATION_STICKY

#define GROWL_NOTIFICATION_STICKY CFSTR(
    "NotificationSticky")
Discussion

A boolean controlling whether the notification is sticky. (Optional)

Not necessarily supported by all display plugins

GROWL_NOTIFICATION_CLICK_CONTEXT

#define GROWL_NOTIFICATION_CLICK_CONTEXT CFSTR(
    "NotificationClickContext")
Discussion

A context for the notification for clicking purposes.

This will be passed back to the application when the notification is clicked (not necessarily supported by all display plugins). It must be plist-encodable.

GROWL_APP_REGISTRATION

#define GROWL_APP_REGISTRATION CFSTR(
    "GrowlApplicationRegistrationNotification")
Discussion

The distributed notification name to use for registration

GROWL_APP_REGISTRATION_CONF

#define GROWL_APP_REGISTRATION_CONF CFSTR(
    "GrowlApplicationRegistrationConfirmationNotification")
Discussion

The distributed notification sent to confirm the registration. Used by the prefpane

GROWL_NOTIFICATION

#define GROWL_NOTIFICATION CFSTR(
    "GrowlNotification")
Discussion

The distributed notification name to use for growl notifications

GROWL_SHUTDOWN

#define GROWL_SHUTDOWN CFSTR(
    "GrowlShutdown")
Discussion

The distributed notification name to use to tell Growl to shutdown

GROWL_PING

#define GROWL_PING CFSTR(
    "Honey, Mind Taking Out The Trash")
Discussion

The distribued notification sent to check if Growl is running. Used by the prefpane

GROWL_PONG

#define GROWL_PONG CFSTR(
    "What Do You Want From Me, Woman")
Discussion

The distributed notification sent in reply to GROWL_PING

GROWL_IS_READY

#define GROWL_IS_READY CFSTR(
    "Lend Me Some Sugar; I Am Your Neighbor!")
Discussion

The distributed notification sent when Growl starts up

GROWL_NOTIFICATION_CLICKED

#define GROWL_NOTIFICATION_CLICKED CFSTR(
    "GrowlClicked!")
Discussion

The distributed notification set when a supported notification is clicked. Handled by the GrowlAppBridge.

GROWL_KEY_CLICKED_CONTEXT

#define GROWL_KEY_CLICKED_CONTEXT CFSTR(
    "ClickedContext")
Discussion

Used internally as the key for the clickedContext passed over DNC

GROWL_REG_DICT_EXTENSION

#define GROWL_REG_DICT_EXTENSION CFSTR(
    "growlRegDict")
Discussion

The extension used when registering an app via opening a file on the helper app
