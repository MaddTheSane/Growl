_launchGrowlIfInstalledWithRegistrationDictionary:

Launches GrowlHelperApp and registers. 

+ (
        BOOL) _launchGrowlIfInstalledWithRegistrationDictionary:(
        NSDictionary *)regDict;
Parameter Descriptions
regDict
The dictionary with which to register.
function result
Returns YES if GrowlHelperApp began launching or was already running, NO if Growl isn't installed
Discussion

Launches the GrowlHelperApp if it's not already running, and passes it a registration dictionary. If Growl is turned on in the Growl prefpane, GROWL_IS_READY will be posted to the distributed notification center when Growl is listening for notifications.

launchGrowlIfInstalled

Launches GrowlHelperApp. 

+ (
        BOOL) launchGrowlIfInstalled;
function result
Returns YES if GrowlHelperApp began launching or was already running, NO if Growl isn't installed
Discussion

Launches the GrowlHelperApp if it's not already running. GROWL_IS_READY will be posted to the distributed notification center once it is ready.

Uses +_launchGrowlIfInstalledWithRegistrationDictionary:.
